# 🎯 群衆大戦 - ARMY RUSH

兵士の**大群**を率い、**＋や×のゲート**で仲間をどんどん増やし、**全員で銃を撃ちまくって**敵バリアとボスをなぎ倒しながら前進し続ける——**群衆シューター**。

「Top War（トップウォー）」や「Count Masters」系の、**仲間が増える快感 × 撃ちまくる爽快感**を一画面に凝縮しました。

🎮 **Web で今すぐ遊ぶ:** https://syoudai0514.github.io/War-game/

---

## 遊びかた

1. **指で部隊を左右にドラッグ**して動かす
2. **＋N / ×2** などの<b>緑のゲート</b>を通ると**仲間が増える**（赤い −N / ÷2 は減るので避ける!）
3. 兵士は**自動で前方に発砲**。人数が多いほど**火力もアップ**
4. **武器UPゲート**で 🔫ピストル → SMG → ライフル → ショットガン → ミニガン → バズーカ → 🛸レーザー と強化
5. 道中の**数字バリア**は撃って壊すか避ける。**ボス👑**は撃ち尽くせ!
6. 倒れたら**コインで永続強化**（開始兵士・火力・連射・開始武器）→ さらに奥へ

距離（m）とコイン・強化はすべて端末に自動保存。エンドレスでベスト距離を更新し続けよう。

---

## 技術構成

- **ビルド不要の素の Web アプリ**（HTML5 Canvas + ES Modules / 依存ライブラリゼロ）
- GitHub Pages はリポジトリ直下をそのまま配信
- Android パッケージは Capacitor でラップ（Google Play 対応・`RELEASE.md` 参照）

```
index.html      … エントリ(HUD・スタート・強化ショップ)
css/style.css   … モバイル縦持ち UI
js/
  config.js     … 武器・ゲート・敵・経済のチューニング値
  audio.js      … WebAudio 効果音
  storage.js    … コイン・ベスト距離・強化のセーブ
  game.js       … ゲーム本体(操舵・ゲート・射撃・敵・ボス・経済)
  render.js     … Canvas 描画
  main.js       … 起動・ループ・操作・HUD・ショップ連携
src/native/main.js … Android 版のみの起動スクリプト(AdMob)
```
